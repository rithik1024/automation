# AWS Configurations Required

- To run terraform create service account : svc-terraform-eks-deploy 
- Create a role with json file "svc-terraform-eks-deploy-aws-role.json"
```
aws iam create-role --role-name TfEksDeployRole  --assume-role-policy-document file://"svc-terraform-eks-deploy-aws-role.json"
```
- Create a policy with the json file "eksclusterpolicy.json"
- Attach policy to the role "TfEksDeployRole"
```
aws iam attach-role-policy --policy-arn arn:aws:iam::058264408282:policy/eksclusterpolicy --role-name TfEksDeployRole
```
